# Política de Privacidade

Última atualização: 2025.02.01

Bem-vindo à Sky Bridge API! Sua privacidade é importante para nós. Esta Política
de Privacidade explica como coletamos, usamos e protegemos suas informações ao
utilizar nossos serviços.

## 1. Informações que Coletamos

A Sky Bridge API pode coletar os seguintes tipos de informações:

### 1.1. Informações Fornecidas pelo Usuário

Quando você usa nossos serviços, podemos coletar informações como:

- Nome do repositório e proprietário (ao buscar commits do GitHub).
- Textos enviados para verificação de segurança.

### 1.2. Informações de Uso

Podemos coletar dados sobre como a API é acessada e usada, incluindo:

- Endereço IP.
- Tipo de dispositivo e navegador.
- Data e hora das requisições.

## 2. Como Usamos as Informações

Usamos as informações coletadas para:

- Fornecer e operar a API.
- Melhorar nossos serviços e segurança.
- Diagnosticar problemas técnicos.
- Prevenir abusos e violações de segurança.

Não vendemos, alugamos ou compartilhamos suas informações com terceiros para
fins comerciais.

## 3. Compartilhamento de Informações

Podemos compartilhar informações apenas nos seguintes casos:

- Cumprimento legal: se exigido por lei, ordem judicial ou solicitação governamental.
- Proteção da API e dos usuários: para detectar e prevenir fraudes, abusos ou
  ameaças à segurança.

## 4. Segurança dos Dados

Adotamos medidas para proteger suas informações contra acessos não autorizados e
vazamentos. No entanto, nenhum sistema é 100% seguro. Recomendamos que você
evite enviar dados sensíveis ao usar a API.

## 5. Seus Direitos e Opções

Você pode:

- Solicitar a exclusão dos seus dados armazenados pela API.
- Revogar o uso da API a qualquer momento.
- Entrar em contato para dúvidas sobre privacidade pelo e-mail: h4mnsoft@gmail.com.

## 6. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos
você sobre mudanças significativas por meio do nosso site ou API.
