# Inbox - Como Usar em 3 Passos

## Passo 1: Capture

Use o comando `/inbox` com uma ideia:

**Claude Code:**
```
/inbox add Adicionar dark mode no Paper Trading UI
```

**Discord:**
```
/inbox add Adicionar dark mode no Paper Trading UI
```

## Passo 2: Confirme

O sistema criará uma issue no Linear com:
- ✅ Título truncado (se > 200 chars)
- ✅ Labels automáticos (fonte, domínio, ação)
- ✅ Descrição estruturada
- ✅ Expires em 60 dias

## Passo 3: Triagem (Semanal)

Revise o Inbox no Linear e aja:
- **Implementar** → Transforme em issue real ou mova para projeto
- **Pesquisar** → Crie doc em `docs/research/`
- **Arquivar** → Mova para "Archived"
- **Descartar** → Delete

---

**Dica:** Use o Inbox para capturar rápido, processe depois. Não deixe ideias valiosas se perderem!
