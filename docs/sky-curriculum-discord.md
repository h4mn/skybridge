### 2. Paper Trading Bot (DDD Architecture)
**Descrição:** Bot de trading simulado com arquitetura orientada a eventos.

**Padrões Aplicados:**
```
Interaction → Command → Handler → Domain Event → Response
```

**Camadas:**
- **Infrastructure:** MCPButtonAdapter (converte Interaction → Command)
- **Application:** ButtonClickHandler (lógica de negócio)
- **Presentation:** PaperTradingBot (respostas Discord)

**Resultado:**
Código escalável, testável, mantível.

### 3. TradingView Multi-Step

**Features:**
